# Thread Opener

Автоматическое открытие закрытых тем на [lolz.live](https://lolz.live) через официальный API.

## Возможности

- Поиск всех закрытых тем с пагинацией
- Массовое открытие через `PUT /threads/{id}`
- Разделение на доступные и без прав (пропускаются)
- Сохранение токена в `config.json`
- Rate limit — 0.35с между запросами
- Цветной вывод: жёлтый (закрыта), зелёный (открыта), красный (нет прав)

## Установка

```bash
pip install requests
```

## Использование

```bash
python thread_opener.py
```

При первом запуске введите API токен — он сохранится в `config.json`.

Получить токен: [lolz.live/account/api](https://lolz.live/account/api)

## Как работает

1. Авторизуется по токену
2. Сканирует все ваши темы (`GET /threads?tab=mythreads`)
3. Находит закрытые (`thread_is_closed: true`)
4. Показывает список и спрашивает подтверждение
5. Открывает каждую (`PUT /threads/{id}`)

> Темы, закрытые модераторами, будут пропущены — на них нет прав редактирования.

## Требования

- Python 3.7+
- requests
